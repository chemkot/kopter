# python
Файлы
* boardDevices.py - реализует классы для управления устройствами на коптере (лидар, интрос, камера, колесная платформа). Работает через VirtualCOMport'ы, опознает устройства по серийным номерам.
* boardDevices_module_tester.py - отдельный скрипт для отладки и тестирования модуля boardDevices.py, можно использовать на земле под Windows
* cmdAgent.py - реализует класс для обмена командой и результатами выполнения между главным потоком и потоком TCP сервера 
* main.py - главный файл программы коптера
* process_controller.py - программа, запускающая main.py и перезапускающая его при падениях и ошибках; пишет вывод в лог-файлы с именем вида launch_YYYY-MM-DD.txt
