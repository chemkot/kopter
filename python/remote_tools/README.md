# python
Папки
* pult.exe - выводит изображение с платы видеозахвата, позволяет управлять коптером (через кнопки на экране)
* comands_sender.exe - простая программа отправки команд по TCP на сервер коптера
* utils - скрипты для обновления ПО коптера (скриптов сервера) по FTP, для скачивания лога, для обработки нажатий клавиш
Файлы
* update_copter_scripts.bat - вызывает Python-скрипт из папки utils, Обновляющий ПО коптера (предварительно нужно прописать в одном из .py файлов в папке utils пути к файлам, порты FTP, IP)
* download_log.bat - вызывает Python-скрипт из папки utils, скачивающий лог вывода main.py коптера текущего дня (предварительно нужно прописать в одном из .py файлов в папке utils пути к файлам, порты FTP, IP)
