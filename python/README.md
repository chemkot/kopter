# python
Папки
* copter_server - код для коптера, логи запусков
* remote_tools - утилиты и программы для планшета - для управления коптером, для обновления ПО коптера, для скачивания логов
* old_code - ненужные куски кода, которые могут пригодиться
** sender.py - примитивный скрипт для подключения к серверу коптера и отправки одной текстовой команды
** enode_example.py - код образца использования ноды в ROS для отправки в ROS положения сервы лидара
* интерфейс интерпретатора - описание команд TCP-сервера коптера (copter_server/main.py). Управление электроникой копетера происходит через эти команды. В данный момент реализованы не все.
* интерфейс супервайзера - описание команд второго TCP-сервера коптера (предназначен для управления загружаемыми миссиями и для аварийной остановки). В данный момент не используется, не все команды реализованы
